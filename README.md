# Ukraine Air Defense Strategy

A browser-based strategy game built with HTML, CSS, and JavaScript.

The player protects Ukrainian cities from incoming missile attacks by strategically placing air defense systems on the map. Different cities have varying levels of importance, and resources are limited, requiring careful planning and quick decision-making.

## Features

* Interactive map of Ukraine
* City protection mechanics
* Missile interception system
* Strategic placement of air defense units
* Score tracking based on saved cities and intercepted missiles
* Built with pure HTML, CSS, and JavaScript

## Technologies

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## Goal

Defend as many cities as possible and achieve the highest score by efficiently deploying air defense systems and intercepting incoming threats.
